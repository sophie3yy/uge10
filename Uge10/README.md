# Week 10 – Review Problems

- **ArithmeticOperations** – Reads two integers and prints their sum, product, difference, quotient and remainder.
- **MultiplesOfNumbers** – Reads two numbers and determines whether the first is a multiple of the second using the modulo operator.
- **DigitSumForeach** – Reads an integer as a string and calculates the sum of its digits using a foreach loop.
- **DigitSumModulo** – Reads an integer and calculates the sum of its digits using modulo and repeated division by 10.
- **DecisionBasedOnBooleans** – Asks three yes/no questions and decides whether you should go to the lecture based on the answers. 
- **SentenceFromWords** – Reads words one by one and prints a complete sentence when the user enters a period (.).
